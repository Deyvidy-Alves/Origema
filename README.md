# Origema 🌱



Site institucional de apresentação da **Origema**, uma empresa fictícia focada em soluções de design sustentável e ecológico. 



Este projeto frontend foi desenvolvido como parte das atividades práticas da disciplina de Design e Multimídia do IF, com o objetivo de criar e aplicar a identidade visual de uma marca na web.



## 🚀 Tecnologias Utilizadas



O projeto foi construído utilizando as seguintes ferramentas:

- **React**

- **Vite**

- **Tailwind CSS**

- **Lucide Icons**



## ⚙️ Como executar o projeto na sua máquina



Certifique-se de ter o Node.js instalado. Siga os passos abaixo:



1. Clone o repositório ou baixe os arquivos.

2. Abra o terminal na pasta do projeto e instale as dependências e inicie o servidor:

```bash

npm install

npm run dev
