# Origema 🌱 | Design Sustentável

> Site institucional e catálogo de produtos de uma empresa de design ecológico, focado em sustentabilidade, economia circular e trabalho artesanal.

Este projeto frontend foi desenvolvido como parte das atividades práticas da disciplina de **Design e Multimídia do IF**, com o objetivo de criar, estruturar e aplicar a identidade visual completa de uma marca fictícia na web.

---

## ✨ Funcionalidades e Destaques

- **Interface Responsiva:** Design que se adapta perfeitamente a celulares, tablets e desktops (Mobile-First).
- **Navegação SPA (Single Page Application):** Troca de páginas rápida e fluida utilizando `react-router`, acompanhada de um script de *Scroll-to-Top* automático.
- **Formulário de Contato Funcional:** Integração com a API do **FormSubmit** para envio real de e-mails direto para a caixa de entrada, sem a necessidade de um backend próprio.
- **Identidade Visual Consistente:** Aplicação rigorosa da paleta de cores da marca (Tons terrosos e verdes) e tipografia mista (Playfair Display para títulos elegantes e Montserrat para leitura clara).
- **Componentes Modernos:** Uso da biblioteca `shadcn/ui` customizada para criar botões, inputs e formulários com visual premium e acessível.

---

## 🗺️ Estrutura do Site

O site é composto pelas seguintes páginas:
- `Início`: Banner de impacto, diferenciais da marca, vitrine de produtos e chamadas para ação.
- `Produtos`: Catálogo completo com filtros por categoria (Decoração, Mobiliário, Utilitários).
- `A Origema (Sobre)`: História da fundação, linha do tempo, missão, visão e valores da empresa.
- `Contato`: Formulário de comunicação direta e links dinâmicos para WhatsApp, E-mail e Google Maps.

---

## 🚀 Tecnologias Utilizadas

A stack principal do projeto inclui:

- **React** + **Vite**: Para uma construção de interface rápida e modular.
- **Tailwind CSS**: Motor de estilização principal do projeto.
- **Shadcn/ui**: Componentes de interface baseados em Radix UI.
- **React Router**: Gerenciamento de rotas da aplicação.
- **Lucide React**: Biblioteca de ícones moderna e leve.
- **FormSubmit**: Serviço de endpoint para o formulário de contato.

## ⚙️ Como executar o projeto na sua máquina

Certifique-se de ter o Node.js instalado no seu computador. Em seguida, abra o seu terminal e siga os passos abaixo:

1. Clone este repositório:
```bash
git clone [https://github.com/Deyvidy-Alves/Origema.git](https://github.com/Deyvidy-Alves/Origema.git)
```

2. Acesse a pasta do projeto:

```bash
cd Origema
```

3. Instale as dependências necessárias:

```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

5. Abra o seu navegador e acesse o link gerado no terminal
